# 🔥 TrainingJJB - Générateur de séances cardio pour grapplers

Une petite app Streamlit qui te génère des séances d'assault bike ou rameur pour booster ton cardio en JJB (nogi).

## ⚙️ Fonctionnalités
- Choix entre assault bike ou rameur
- Sélection du type de séance (HIIT explosif ou lactique)
- Génération aléatoire ou manuelle

## 🚀 Lancer l'app
```bash
streamlit run app.py
